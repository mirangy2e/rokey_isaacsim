📷 Z축 = 렌즈 방향(앞쪽). Y-up 뷰에서 Y(Pan)를 돌리면 카메라가 좌우로 스윙하고, X(Tilt)를 올리면 하늘·땅을 바라봅니다. Z-Roll은 수평선이 기울 때 씁니다.
두 회전 순서 한눈에 비교

오브젝트 / 로봇

Z → Y → X
Z = Yaw (방향 전환)
Y = Pitch (앞뒤 기울기)
X = Roll (좌우 기울기)
주행로봇, 매니퓰레이터, 일반 오브젝트
Z-up 뷰 / Z가 위

카메라

Y → X → Z
Y = Pan (좌우 스윙)
X = Tilt (상하 각도)
Z = Roll / 렌즈 방향
카메라 센서, 뷰포트 카메라
Y-up 뷰 / Z가 앞(렌즈)
구분 ZYX (로봇) YXZ (카메라)
Up 축Z (위)Y (위)
렌즈/앞 방향XZ
1순위 회전 Z Yaw Y Pan
Gimbal LockY = ±90° 시X = ±90° 시

⚠️ 실습 주의: Properties에서 Euler로 회전값 입력 시 ZYX 순서가 적용됩니다. 예상과 다른 방향으로 움직이면 순서를 먼저 확인하세요.

📌 실무 팁: 복잡한 회전은 Quaternion으로 지정하면 회전 순서 문제를 원천 차단할 수 있습니다.
